# V13_CHANGELOG

## v13 초보자/멀티소재 직접 캐릭터 만들기 모드

- 보리/쌀, 감자/고구마처럼 2개 소재 듀오 캐릭터 입력 지원
- 사용자가 정한 기본 도형, 색상, 성격, 말투를 기반으로 360×360 기본 시안 생성
- 보리/쌀처럼 서로 다른 성격과 말투를 가진 듀오형 표현 80개 자동 생성
- 표현 PNG 미리보기 자동 생성
- 움직이는 문구 결합형 장면 설계표 자동 생성
- 레이어 후보 PNG, 창작 타임라인, SHA-256 체크섬 기록
- 초보자 창작 리포트 HTML/JSON/ZIP 생성

주의: 이 기능은 AI 완성본을 숨기거나 검수 우회하는 기능이 아니라, 사용자의 직접 창작 출발점을 기록하고 상품화 작업을 보조하는 기능입니다.
