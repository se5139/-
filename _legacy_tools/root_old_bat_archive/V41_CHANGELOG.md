# V41_CHANGELOG

## v41: 선택된 1차 포맷 자동 변환/압축 엔진

v41은 v37/v40에서 결정된 1차 제작 포맷 1개만 대상으로 규격 맞춤/용량 압축/아이콘·공유 이미지 생성을 수행합니다.

### 핵심 원칙
- 처음부터 모든 포맷을 동시에 만들지 않는다.
- 선택된 1차 포맷만 fixed 폴더에 제출 후보로 만든다.
- 원본은 original 폴더에 보존하고 덮어쓰지 않는다.
- 미니/큰/움직이는 포맷은 데이터가 쌓인 뒤 확장 후보로만 검토한다.

### 추가 기능
- 선택 포맷별 수량/크기/용량 기준 자동 적용
- PNG 자동 360x360/180x180/540x540 등 맞춤
- PNG 용량 최적화
- 움직이는 포맷 WebP 생성/프레임 수 조정
- 아이콘 78x78 자동 생성
- 공유 이미지 600x166 자동 생성
- 수정 전/후 CSV 리포트 생성
- HTML/JSON/TXT/ZIP 리포트 생성
