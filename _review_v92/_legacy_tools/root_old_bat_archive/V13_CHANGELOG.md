# v13 변경사항 - 색상/성격/말투 후보 선택 + 실시간 미리보기

## 추가 기능

- 초보자/멀티소재 직접 만들기 화면에 색상 후보 팔레트 추가
  - 따뜻한 곡물톤
  - 부드러운 파스텔톤
  - 선명한 캐릭터톤
  - 흑백 낙서톤
  - 직장인 차분톤
  - 과일·채소톤
  - 밤하늘 감성톤
- 소재별 색상 선택 방식 추가
  - 전체 팔레트 자동
  - 소재별 후보 선택
  - 직접 입력
- 소재별 성격 후보 선택 추가
- 소재별 말투 후보 선택 추가
- 소재별 역할 후보 선택 추가
- 선택값 기반 360×360 실시간 미리보기 추가
- 선택값 기반 표현 후보 일부 미리보기 추가
- 움직이는 문구형 장면 후보 미리보기 추가

## 유지 기능

- 도형으로 직접 캐릭터 만들기 유지
- 스케치/사진/파일 첨부 기능 유지
- 단어 입력 기능 유지
- 최대 5개 소재 조합 유지
- 첨부 이미지는 직접 합성/복제하지 않고 색감·형태·비율 힌트와 창작 증거로만 사용

- 표현 후보 80개 생성 PASS
- 리포트/ZIP 생성 PASS
