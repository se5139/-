카카오 이모티콘 수익화 시스템 v36 요약

이번 버전은 카카오 이모티콘 규격/용량 최종 검수 엔진을 추가한 버전입니다.
사용자가 업로드한 규격 영상에서 확인한 포맷별 수량·파일 형식·사이즈·용량·프레임·반복·공유 이미지·아이콘·사운드 조건을 프로그램 검수 탭에 반영했습니다.

추가 탭:
- 36 카카오 규격/용량 검수

주요 기능:
- 포맷 선택: 멈춰있는 / 움직이는 / 큰 / 멈춰있는 미니 / 움직이는 미니
- 검수 폴더 지정
- 파일명/폴더명 기반 item/icon/share/sound 자동 분류
- 수량 검사
- 확장자 검사
- 픽셀 크기 검사
- 용량 검사
- WebP 프레임 수 검사
- 투명 배경/알파 채널 검사
- 다크모드 대비, 마지막 프레임, 미니 순서 배치, 사운드 아이콘 가림 위험 수동 확인 메모
- HTML/JSON/CSV/TXT/ZIP 리포트 생성

주의:
이 기능은 제출 전 사전 검수용입니다. 카카오 공식 가이드는 변경될 수 있으므로 최종 제출 직전에는 카카오 이모티콘 스튜디오 최신 기준을 반드시 다시 확인해야 합니다.
