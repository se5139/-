# V25_CHANGELOG

## v25 Drawing Refine + Parts Estimate

이번 버전은 v24의 자유 드로잉 캔버스 결과를 실제 이모티콘 제작 흐름으로 더 강하게 연결합니다.

### 추가 기능

- 자유 드로잉 원본 360×360 자동 정리
- 얼굴/몸통/눈/입/팔 후보 파츠 추정
- 파츠 추정 오버레이 PNG 생성
- 파츠별 PNG 분리 저장
- 기본/웃음/감사/사과/화남/놀람/피곤/확인 등 표정 변형 PNG 생성
- 24개/32개 세트 제작 연결용 스타터 표현 CSV/JSON 생성
- v25 리포트/ZIP 생성
- 기존 기능 유지: v1~v24 전체 기능, 데이터 보호/백업, 성장형 학습, 자유 드로잉 캔버스

### 설계 원칙

- 기존 파일을 덮어쓰지 않고 별도 결과 폴더에 저장합니다.
- 파츠 추정은 초보자 제작 보조용이며, 최종 판단은 v16 표정/파츠 편집기와 v23 일관성 검사에서 다시 확인합니다.
- AI 완성본 은폐가 아니라 사용자가 직접 그린 원본을 정리/확장하는 구조입니다.
