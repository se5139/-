이번 버전은 유튜브 참고영상/자막 분석기를 추가했습니다.
URL 또는 iframe에서 videoId를 추출하고, 사용자가 붙여넣은 자막 또는 TXT/SRT/VTT 파일을 분석해 이모티콘 승인/수익화 관련 핵심 주장, 위험 주장, 안전한 기능 반영 후보를 분리합니다.
YouTube API 키가 있으면 영상 제목/설명/통계/댓글도 선택적으로 조회할 수 있으며, 영상 파일을 다운로드하거나 재사용하지 않습니다.
AI 은폐, 검수 우회, 기존 캐릭터 모방, 무단 재사용 관련 주장은 프로그램 기능으로 구현하지 않고 위험 경고/차단 항목으로 기록합니다.
