# v34 변경사항 — 구체 콘셉트·멘트·모션 템플릿 전략 엔진

- 유튜브 참고자료 "카톡 이모티콘 만들고 월 1200만원 버는 30대 여자"의 핵심 구조를 안전한 제작 전략으로 반영했습니다.
- 막연한 "귀여운 고양이"식 입력을 감지하고, 성격·상황·말투가 드러나는 구체 콘셉트로 보정합니다.
- 제목 후보 10개를 생성하고, 제목에 소재/성격/사용상황이 드러나는지 점수화합니다.
- 그림 제작 전에 24개 또는 32개 문구를 선기획합니다.
- 이모티콘 플러스 검색/추천을 고려해 짧고 검색 가능한 키워드 후보를 생성합니다.
- 손흔들기, 꾸벅, 박수, 따봉, 도장/체크, 눈물 등 모션 템플릿 라이브러리를 추가했습니다.
- 정지형/문구형/움직이는 문구형/미니/움직이는 이모티콘 중 포맷 추천을 콘셉트·문구·사용성 기준으로 계산합니다.
- 수익 보장 또는 승인 보장이 아니라, 제작 전 전략 정리와 위험도 낮추기 목적입니다.
