# V13_CHANGELOG

## v13 도형/스케치/첨부파일 기반 멀티소재 캐릭터 생성

추가 기능:

- 도형으로 캐릭터 직접 만들기 흐름 유지
- 사용자가 직접 그린 스케치 사진, PNG/JPG/WEBP/BMP 이미지, TXT/JSON 설명 파일 첨부 가능
- 첨부 자료는 그대로 합성/복제하지 않고 색감·비율·형태 힌트와 창작 증거로만 기록
- 2개 듀오형뿐 아니라 1~5개 소재 멀티 캐릭터 생성 지원
- 소재별 이름, 기본색, 성격, 말투, 기본 도형, 역할 입력
- 멀티 소재 기본형 360×360 PNG 생성
- 소재별 단독 후보 PNG 생성
- 멀티소재 표현 후보 40~120개 생성
- 움직이는 문구형 장면 설계표 생성
- 첨부자료 SHA-256 체크섬/창작 과정 타임라인 기록
- `multi_material_creator_report.html/json` 및 ZIP 패키지 생성

주의:

- 이 기능은 AI 완성본 은폐/검수 우회 기능이 아닙니다.
- 첨부 이미지는 사용자가 직접 만든 자료나 직접 촬영/제작한 자료를 전제로 하며, 기존 캐릭터 이미지를 따라 만들기 위한 용도로 사용하지 않습니다.
