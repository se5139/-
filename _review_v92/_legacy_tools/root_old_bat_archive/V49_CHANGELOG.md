# v49 변경사항

## 핵심 목표
v48의 후보 적용/정지형 품질 진화 흐름을 확장하여, 정지형 캐릭터와 움직이는 캐릭터를 함께 개선하는 통합 품질 진화 단계입니다.

## 추가/개선 기능

1. 후보 적용/정지형/움직이는형 품질 진화 탭 추가
   - 후보를 화면에 보여주는 데서 끝내지 않고 현재 제작 흐름에 적용합니다.
   - 적용 대상: active generation profile, text prompt, expression bank, candidate gallery, motion plan.

2. 정지형 캐릭터 품질 진화
   - 실루엣, 외곽선, 표정 대비, 포즈 다양성, 문구 가독성, 채팅창 인식성을 분석합니다.
   - 참고자료의 캐릭터 외형을 복제하지 않고 추상 품질 신호만 사용합니다.

3. 정지형 → 움직이는형 변환 계획
   - 정지형에서 만든 캐릭터 정체성, 색상, 외곽선, 얼굴 비율, 말투를 유지합니다.
   - 움직이는형은 정지형 시안을 기준 프레임으로 삼아 눈깜빡임, 통통 움직임, 말풍선 팝업, 꾸벅임 등을 적용합니다.

4. 다중 파일/ZIP 업로드 분석
   - PNG/JPG/WEBP/BMP/GIF/TXT/MD/CSV/JSON/SRT/VTT/ZIP 다중 업로드를 지원합니다.
   - ZIP 내부 자료는 안전 경로로 추출 후 이미지/텍스트/프레임/색상 정보를 추상 분석합니다.

5. 화면 UI 개선
   - v49 탭에 카드형 UI, 단계형 진행 박스, 대시보드형 지표, 큰 실행 버튼을 적용했습니다.
   - 초보자가 다음 단계가 무엇인지 더 쉽게 보도록 구성했습니다.

6. 안전장치
   - 기존 캐릭터/상표/고유 디자인 복제 금지.
   - YouTube/인터넷 자료는 사용자가 입력한 메모, 업로드 파일, 공식 API 기반 자료를 우선합니다.
   - 무단 크롤링/검수 우회/AI 은폐 목적 기능은 넣지 않습니다.
