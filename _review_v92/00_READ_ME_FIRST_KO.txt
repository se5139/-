먼저 읽어주세요 - 카카오톡 이모티콘 직접 제작 도우미 v92

1. ZIP을 완전히 압축 해제하세요.
2. 47_V92_ERROR_DIAGNOSTICS_OUTPUT_CHECK.bat 로 전체 점검을 먼저 실행하세요.
3. 00_STEP_2_PORTABLE_INSTALL_NOW.bat 로 포터블 설치를 진행하세요.
4. 00_STEP_3_START_PROGRAM.bat 로 프로그램을 실행하세요.

v92 핵심:
- v91의 간편 PNG/GIF 출력 구조 유지
- 정지형 PNG 32개, 움직이는 GIF 24개 생성
- JPG는 확인용 미리보기만 사용
- 제출용 ZIP에는 JPG/JPEG 미포함
- 오류 ID 기반 안전 오류진단/로그 기능 추가
- API 키/비밀번호/토큰 형태 값 마스킹
- 로그는 사용자 데이터 폴더에 저장
- 기본 메뉴 5개 큰 흐름 유지

Windows 설치마법사 EXE는 이 환경에서 직접 컴파일하지 못했습니다.
사용자 Windows PC에서 Inno Setup 설치 후 00_DOUBLE_CLICK_THIS_FIRST_BUILD_SETUP_EXE.bat 를 실행해야 합니다.
