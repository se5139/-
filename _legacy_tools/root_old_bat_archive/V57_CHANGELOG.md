# V57 Changelog - Sidebar Navigation UI

- 기존 상단 가로 탭 49개를 좌측 세로 메뉴 방식으로 전환했습니다.
- 메뉴 검색 입력창을 추가해 `움직이는`, `API`, `품질`, `정지형` 같은 키워드로 기능을 빠르게 찾을 수 있습니다.
- 너무 길었던 상단 설명 문구를 짧게 줄이고, 전체 기능 이력은 접이식 영역으로 이동했습니다.
- 기존 49개 기능 블록은 삭제하지 않고 `selected_page_index` 라우팅 방식으로 유지했습니다.
- 최종 프로그램 기준은 Python/Streamlit 로컬 실행 방식입니다.
