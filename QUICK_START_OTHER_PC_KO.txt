Kakao Emoticon Maker v100 Clean - 다른 PC 실행 순서

1. GitHub 또는 전달받은 파일에서 아래 ZIP을 받습니다.
   release/kakao_emoticon_v100_clean_latest.zip

2. ZIP 압축을 풉니다.
   압축을 풀면 kakao_emoticon_v100_clean 폴더가 생깁니다.

3. 먼저 VERIFY_PACKAGE.bat을 실행합니다.
   모든 항목이 OK면 패키지가 정상입니다.

4. START_WINDOWS.bat을 실행합니다.
   첫 실행은 Python 가상환경을 만들고 Pillow를 설치하므로 시간이 조금 걸릴 수 있습니다.

5. 브라우저가 열리지 않으면 아래 주소를 직접 엽니다.
   http://127.0.0.1:8520

주의:
- Python 3.10 이상이 필요합니다.
- 첫 실행 때만 인터넷 연결이 필요할 수 있습니다. 인터넷은 Pillow 설치에 사용됩니다.
- API 키가 없어도 기본 제작, 검수, 자료 메모 분석은 가능합니다.
- 결과물은 outputs 폴더에 생성되며, portable ZIP에는 기본 포함되지 않습니다.
