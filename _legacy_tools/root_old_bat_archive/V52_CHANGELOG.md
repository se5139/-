# v52 정지형 기반 움직이는형 생성 / 선택 제안 반영 재생성

## 핵심 변경
- 정지형 캐릭터 생성 후 동일 identity를 잠가 움직이는형 GIF 초안 생성.
- 사용자가 선택한 개선 제안을 실제 정지형 PNG 재생성과 GIF 생성 규칙에 반영.
- 생성 결과를 현재 제작 흐름(prototype_results, expressions, last_gif)에 적용하는 버튼 추가.
- 새 탭: `49 정지형 기반 움직이는형/제안 반영`.

## 안전 원칙
- 기존 캐릭터 복제 금지.
- 선택 제안은 외곽선, 표정 대비, 시그니처 포즈, 말풍선, 모션 동기화 같은 추상 제작 속성만 반영.
- 원본/이전 결과 덮어쓰기 금지.
