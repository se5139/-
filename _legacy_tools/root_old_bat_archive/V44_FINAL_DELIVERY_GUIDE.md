# 카카오 이모티콘 수익화 시스템 v44 최종 실행 가이드

## 1. 가장 쉬운 실행 순서

1. ZIP 압축을 해제합니다.
2. `1_INSTALL_NOW.bat`을 실행합니다.
3. 설치가 끝나면 `2_START_PROGRAM.bat` 또는 `START_WINDOWS.bat`을 실행합니다.
4. 문제가 있으면 `10_V43_DIAGNOSE_AND_REPAIR.bat` 또는 `12_V44_FULL_INTEGRATION_CHECK.bat`을 실행합니다.

## 2. v44 검증 실행

`12_V44_FULL_INTEGRATION_CHECK.bat`을 더블클릭하면 아래 항목을 확인합니다.

- Python 파일 컴파일
- 핵심 모듈 import
- 실행/복구/백업 BAT 파일 존재 여부
- 사용자 데이터 분리 구조
- 카카오 스튜디오 엑셀 업로드 학습 흐름
- 성과 대시보드 흐름
- 1차 포맷 추천 흐름
- 선택 포맷 자동 보정 흐름
- 타 플랫폼 재패키징 초안 흐름
- 백업 ZIP 생성 및 검증

검증 결과는 `outputs/v44_final_integration/` 아래에 HTML, JSON, CSV, TXT로 저장됩니다.

## 3. 중요한 운영 원칙

이 프로그램은 이모티콘 제작을 보조하는 로컬 PC 프로그램입니다. 카카오 승인이나 수익을 보장하지 않습니다.

- 직접 그린 스케치, 도형, 문구, 수정 기록을 중심으로 창작 증거를 남기세요.
- 기존 캐릭터, 상표, 유명 IP, 연예인 얼굴을 따라 하지 마세요.
- 카카오/네이버 OGQ/LINE/BAND 등 공식 규격은 제출 직전에 다시 확인하세요.
- 업데이트 전에는 `8_BACKUP_USER_DATA.bat` 또는 v44 통합 검증의 백업 기능으로 데이터를 보존하세요.
- 사용자 승인 없는 데이터 삭제/덮어쓰기는 하지 않는 구조를 유지합니다.

## 4. v44에서 검증하는 핵심 연결 흐름

```text
카카오 스튜디오 엑셀 2종 업로드
→ v39 성과 학습
→ v40 대시보드/다음 제작 방향 추천
→ v37 1차 포맷 추천
→ v41 선택된 1개 포맷만 자동 보정
→ v42 타 플랫폼 재패키징 초안
→ v44 최종 검증 리포트
```

## 5. 공식 기준 확인 필요 표시

프로그램 안의 규격표는 제작 보조 기준입니다. 실제 제출 기준은 바뀔 수 있으므로 최종 제출 전 공식 스튜디오에서 다시 확인해야 합니다.
