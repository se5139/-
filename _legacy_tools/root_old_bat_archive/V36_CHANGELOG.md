# V36 변경사항 - 카카오 규격/용량 최종 검수 엔진

## 핵심 추가
- `36 카카오 규격/용량 검수` 탭 추가
- 업로드된 규격 영상과 카카오 공개 가이드 기준을 바탕으로 포맷별 사전 검수 기능 추가
- 멈춰있는/움직이는/큰/미니/움직이는 미니 이모티콘 규격 테이블 내장
- 폴더 내 파일을 item/icon/share/sound로 자동 분류
- PNG/WebP/GIF/MP3 용량·크기·개수·프레임·투명 배경 위험 검사
- WebP 최소 제출 개수, 프레임 수, 마지막 프레임/반복/다크모드 대비 수동 확인 메모 출력
- HTML/JSON/CSV/TXT/ZIP 규격 검수 리포트 생성

## 반영한 영상 핵심
- 멈춰있는 이모티콘: 32개 PNG, 360×360, 150KB 이하, 아이콘 78×78, 공유 이미지 600×166
- 움직이는 이모티콘: 24개, WebP 3개 이상, 360×360, 650KB 이하, 24프레임 이하, 4회 반복
- 큰 이모티콘: 16개, 540×540/300×540/540×300, 1MB 이하
- 미니 이모티콘: 멈춰있는 42개 180×180 PNG 100KB 이하, 움직이는 35개 180×180 WebP/PNG 500KB 이하
- 공통: 72dpi, RGB, 투명 배경, 다크모드 대비, 미니 이모티콘 순서/조합성, 사운드형 MP3 5초 이하/200KB 이하
