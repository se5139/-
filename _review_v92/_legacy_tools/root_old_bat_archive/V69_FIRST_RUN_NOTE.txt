Kakao Emoticon Profit System v69

1. 새 폴더에 압축 해제하세요. 예: C:\KakaoEmoticonV69

4. 프로그램 좌측 메뉴에서 56 실제 그림체/모션 품질 고도화를 실행하세요.
5. 이전 버전 정리는 메인 프로그램이 정상 실행되는 것을 확인한 뒤 진행하세요.
