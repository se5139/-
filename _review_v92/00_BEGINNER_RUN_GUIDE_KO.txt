카카오톡 이모티콘 직접 제작 도우미 v92 초보자 실행 안내

이 파일은 처음 실행하는 사용자를 위한 안내입니다.

1. 압축 해제
- ZIP 파일을 새 폴더에 압축 해제하세요.
- 권장 위치: C:\KakaoEmoticonV92
- 압축파일 안에서 바로 실행하지 말고 반드시 압축을 풀고 실행하세요.

2. 먼저 전체 점검 실행
- 47_V92_ERROR_DIAGNOSTICS_OUTPUT_CHECK.bat 를 먼저 실행하세요.
- 이 점검은 PNG 32개, GIF 24개, JPG 미리보기 56개, 제출용 ZIP JPG 제외, 파일명 안전성, 오류진단 로그 구조를 확인합니다.

3. 포터블 설치
- 00_STEP_2_PORTABLE_INSTALL_NOW.bat 를 실행하세요.
- 설치 성공 후 v91 이하 이전 버전 폴더는 사용자 데이터 후보를 백업한 뒤 정리합니다.
- 현재 v92 이상 폴더는 정리하지 않습니다.

4. 프로그램 실행
- 00_STEP_3_START_PROGRAM.bat 를 실행하세요.
- 브라우저에서 Streamlit 화면이 열립니다.

5. 설치마법사 EXE 만들기
- Windows PC에 Inno Setup이 설치되어 있으면 00_DOUBLE_CLICK_THIS_FIRST_BUILD_SETUP_EXE.bat 를 실행하세요.
- 생성 위치: installer\Output\KakaoEmoticonSetup_v92.exe

6. 작업 흐름
기본 화면은 5개 큰 단계만 사용합니다.

1 제작 시작 · 정지형/움직이는형 미리보기
2 세트 구성 · 32개/24개 품질 진화
3 검사 · 자동보정 · 제출 전 승인
4 반려 대응 · 캡처/OCR · 재생성
5 최종 납품 · 백업/리포트/재검사

7. 출력 구조
- 정지형 최종 후보: static_png_submit 폴더의 PNG 32개
- 움직이는형 최종 후보: animated_gif_submit 폴더의 GIF 24개
- 확인용 미리보기: preview_jpg 폴더의 JPG 56개
- 제출용 ZIP: v92_submit_only_png_gif.zip
- 제출용 ZIP에는 JPG/JPEG가 들어가지 않습니다.

8. v92 오류진단 기능
- 오류가 나면 오류 ID가 표시됩니다.
- 로그는 프로그램 폴더가 아니라 사용자 데이터 폴더에 저장됩니다.
- Windows 기준 저장 위치: %LOCALAPPDATA%\KakaoEmoticonProfitSystemV92\logs
- API 키, 비밀번호, 토큰 형태 값은 저장 전 마스킹됩니다.
- 5번 최종 납품 화면의 고급 진단에서 최근 오류/작업/성능 로그와 진단 ZIP을 확인할 수 있습니다.

9. 중요한 안전 원칙
- 자동 제출 기능은 없습니다.
- 기존 인기 캐릭터, 상표, 유명 캐릭터를 복제하지 않습니다.
- 온라인 자료는 원본 복제가 아니라 문구 길이, 감정 유형, 포즈, 모션 리듬 같은 추상 신호만 참고합니다.
- API 키는 선택형이며 유료 API는 기본 OFF입니다.
- 사용자 데이터는 승인 없이 삭제하거나 덮어쓰지 않습니다.
