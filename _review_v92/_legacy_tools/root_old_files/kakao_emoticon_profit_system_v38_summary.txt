카카오 이모티콘 수익화 시스템 v38 요약

이번 버전은 CSV/캡처 기반 데이터 입력 파이프라인을 추가했다.
사용자는 카카오 심사 결과, 반려 사유, 월별 판매/정산 메모, 문구 반응, 품질검사 결과 등을 CSV 템플릿 또는 캡처 이미지로 가져올 수 있다.
프로그램은 원본 파일을 보존하고, 컬럼 자동 매칭/검증/정리 CSV 생성/HTML 리포트/JSON 리포트/성장형 학습 JSONL 누적 저장을 수행한다.
캡처 이미지는 원본과 미리보기를 보존하고, 사용자가 붙여넣은 핵심 메모에서 날짜/상태/숫자/문구 후보를 추출한다.
캡처 기반 데이터는 오차 가능성이 있으므로 반드시 사용자가 확인 후 학습 데이터로 확정하는 구조다.
기존 v37의 1차 포맷 추천/단계별 확장 전략과 v19 데이터 보호 구조는 유지한다.
