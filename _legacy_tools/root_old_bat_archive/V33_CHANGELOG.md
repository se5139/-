# v33 지역 특성·사투리 실생활 문구 엔진

## 추가 기능
- 새 탭 `33 지역·사투리 문구` 추가
- 지역/생활권 선택: 충청권, 강원권, 경상권, 전라권, 제주권, 수도권, 직접 입력
- 사용자 실제 생활 사투리/말투 입력 기능
- 지역형 캐릭터 콘셉트 한 줄 생성
- 제목 후보 10개 생성 및 콘셉트 점수화
- 24~42개 사투리/생활 문구 후보 생성
- 문구별 사용 상황, 캐릭터 메모, 모션 힌트, 안전 메모 생성
- 지역 비하/희화화/고정관념 위험 표현 검사
- 국립국어원 지역어/말뭉치 자료 참고 원칙 메모 생성
- HTML/JSON/CSV/TXT/ZIP 리포트 생성

## 설계 원칙
- 공식/공개 지역어 자료는 검증 참고 구조로 사용
- 원문 대량 복제 없이 사용자 경험 말투와 안전한 후보 템플릿을 결합
- 특정 지역을 웃음거리로 만들지 않고, 생활형 짧은 문구와 개인 경험 기반 독창성을 강화
- 실제 지역/세대/가족·직장 관계별 어색함은 사용자 최종 검수 권장
