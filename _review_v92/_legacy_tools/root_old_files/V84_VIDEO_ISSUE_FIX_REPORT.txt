v84 영상 오류 확인 및 수정 리포트

확인된 영상 내용:
1. 파일 탐색기에서 v83 압축해제 폴더가 열려 있음.
2. 0_BUILD_WINDOWS_INSTALLER_EXE.bat 파일이 선택되어 있음.
3. 영상 내에서는 BAT 실행 후 CMD 로그 또는 오류창이 확인되지 않음.
4. 따라서 영상만으로는 실제 빌드 실패가 아니라, 파일이 선택만 된 상태로 판단됨.

패키지 자체에서 확인한 문제:
1. v83 ZIP 루트에 150개 이상의 BAT 파일이 존재.
2. 구버전 V43~V82 실행/검사/정리 BAT가 루트에 그대로 노출됨.
3. 사용자가 실행해야 할 파일이 묻혀 있어 UX 오류가 발생.

v84 수정:
1. 구버전 루트 BAT를 _legacy_tools/root_old_bat_archive로 이동.
2. 00_DOUBLE_CLICK_THIS_FIRST_BUILD_SETUP_EXE.bat 추가.
3. 00_STEP_1/2/3 실행 순서 파일 추가.
4. v84 root cleanup check 추가.
5. 설치마법사 빌드 스크립트의 안내 문구 강화.
