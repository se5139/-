# v28 변경 사항

## 누락 정보 후보 제안/재구성
- 텍스트 설명에서 빠진 색상, 성격, 말투, 표정, 팔/손, 다리/발, 행동, 움직임, 포맷을 자동 감지합니다.
- 각 누락 항목마다 후보를 보여주고 선택한 값으로 프롬프트를 재구성합니다.
- 사용자가 원하면 후보 보완 없이 “입력 그대로 만들기”로도 초안을 생성합니다.
- 재구성 결과를 360×360 PNG, 움직이는 GIF, CSV/HTML/JSON/ZIP 리포트로 저장합니다.
- 기존 직접 그리기, 텍스트 설명 생성, 감정/모션 확장, 데이터 보호 기능은 유지됩니다.
