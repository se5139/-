# v21 초보자 전체 제작 마법사

## 추가 기능
- `21 제작 마법사` 탭 추가
- 소재 선택 → 색상/성격/말투 → 표현 후보 → 최종 세트 → 표정/파츠 → 채팅창 미리보기 → 품질검사 → 저작권 방어 → 제출 패키지 → 백업 → 성장형 학습 저장 순서 안내
- 원하면 첫 샘플 세트 생성까지 자동 실행
- 마법사 HTML/JSON/CSV/ZIP 리포트 생성
- 기존 v20 성장형 학습/데이터 보호 구조 유지

## 원칙
- 초보자용 순서 안내와 제작 보조 기능이며 카카오 승인·수익 보장이 아님
- 직접 창작 기반, AI 완성본 은폐/검수 우회 기능 없음
