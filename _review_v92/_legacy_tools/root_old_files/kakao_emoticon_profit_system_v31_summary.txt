카카오 이모티콘 수익화 시스템 v31 요약

이번 버전은 최종 제출 패키지 생성 마법사를 추가했습니다.

핵심 기능:
1. v30 제출 전 잠금 체크리스트 통과 여부 확인
2. 통과 시 최종 제출 준비 ZIP 생성
3. 미통과 시 최종 ZIP 생성 차단 또는 내부 확인용 초안 ZIP 생성
4. 선택 포맷별 24개/32개 등 목표 수량 패키지 생성
5. HTML/JSON/CSV/매니페스트/관련 리포트 통합 번들 생성
6. SHA-256 체크섬 생성

주의:
이 기능은 카카오 승인이나 수익을 보장하지 않으며, 제출 전 공식 최신 기준을 반드시 다시 확인해야 합니다.
