
# v65 전문 IDE/코딩 프로그램 적용 기준

최종 납품 파일은 Python 중심 로컬 PC 실행/설치형 프로그램으로 유지한다. 전문 IDE와 보조 코딩 도구는 개발 과정에서만 쓰며, 최종 사용자가 반드시 설치해야 하는 의존성으로 만들지 않는다.

## 즉시 우선 도구
- PyCharm Professional: Python 전문 개발, 디버깅, 테스트, 프로젝트 구조 관리
- Visual Studio Code: 빠른 수정, 확장, JSON/CSV/Markdown/스크립트 관리
- Git: 버전관리, 롤백, 릴리스 태그
- Inno Setup: Windows 설치마법사

## 확장 단계 도구
- Visual Studio: Windows native/C#/C++ 보조가 필요할 때
- WebStorm: React/Next.js 관리자 웹 대시보드가 필요할 때
- Android Studio: 플레이스토어 앱 확장 단계
- Xcode: iOS 앱 확장 단계이며 Mac 필요
- CLion/Rider/IntelliJ/Eclipse/NetBeans: 해당 언어/플랫폼 필요 시 선택
