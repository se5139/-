카카오 이모티콘 수익화 시스템 v24 요약

이번 버전은 자유 드로잉 캔버스 강화판입니다.
사용자는 마우스, 태블릿 펜, 터치 화면 손그림으로 대충 원·눈·입·몸통을 직접 그리고, 프로그램은 이를 360×360 투명 PNG와 자동 정리 PNG, 스트로크 기록, 창작 증거 리포트로 저장합니다.
브라우저 드로잉 패키지가 설치되어 있으면 직접 그릴 수 있고, 설치되지 않은 환경에서는 좌표 입력 또는 샘플 스트로크로도 테스트 가능합니다. 기존 도형/파츠 편집, 멀티소재, 후보 갤러리, 표정 편집, 채팅 미리보기, 일관성 검사, 데이터 보호, 성장형 학습 기능은 유지됩니다.
