# Kakao Safe Creation Workflow

이 문서는 카카오 이모티콘 제안을 준비할 때 v100을 안전하게 쓰기 위한 기준입니다.

## 중요한 판단

대충 스케치를 했다는 사실만으로 자동 생성 결과물이 항상 안전한 제출물이 되는 것은 아닙니다.

안전한 방향은 다음과 같습니다.

- 스케치와 콘셉트는 사람이 직접 만듭니다.
- 자동 생성 결과물은 아이디어 참고, 문구 구성, 규격 점검에만 사용합니다.
- 최종 제출 이미지는 사람이 직접 선화, 채색, 수정, 마감합니다.
- 작업 증빙을 보관합니다.

## 권장 제작 흐름

1. 종이 또는 태블릿에서 러프 스케치를 만듭니다.
2. 캐릭터 비율, 표정, 포즈를 사람이 직접 확정합니다.
3. v100은 문구 구성과 규격 체크에 사용합니다.
4. 자동 생성 이미지는 그대로 제출하지 않습니다.
5. 직접 그린 파일을 v100의 검수/패키징 대상으로 사용합니다.
6. 결과 폴더의 `human_origin_checklist.json`을 함께 보관합니다.

## 제출 전 체크

- 타인 캐릭터, 상표, 유명인, 기존 이모티콘을 따라 하지 않았는가?
- AI 결과물을 거의 그대로 제출하지 않았는가?
- 직접 수정한 단계별 증빙이 있는가?
- 이미지 유형, 개수, 크기, 용량이 맞는가?
- 미니/일반/움직이는 이모티콘 유형을 섞어 제출하지 않았는가?
- 유튜브/인터넷 자료에서 캐릭터, 구도, 문구, 썸네일을 그대로 베끼지 않았는가?

## 현재 v100의 위치

현재 자동 생성 기능은 참고용입니다.

다음 개발 단계에서는 `직접 제작 이미지 업로드 -> 유형별 규격 검사 -> 용량 압축 -> 제출 패키징` 기능을 중심으로 옮겨갑니다.

## 자료 수집 원칙

자료 수집은 프로그램을 진화시키기 위한 추상 학습용입니다.

허용되는 활용:

- 이모티콘 유형과 규격 파악
- 자주 쓰이는 감정 카테고리 파악
- 말투의 길이와 분위기 파악
- 작은 화면에서 잘 보이는 형태적 원칙 파악

피해야 할 활용:

- 특정 작가 캐릭터 복제
- 특정 이모티콘 문구 그대로 복사
- 썸네일, 구도, 동작을 그대로 따라 하기
- 브랜드, 연예인, 기존 IP와 유사하게 만들기

## 자동 수집 원칙

v100의 자동 수집은 다음 자료를 우선 포함합니다.

- 카카오 이모티콘 스튜디오 운영 원칙
- 카카오 FAQ 및 약관
- 카카오 WebP 애니메이터
- 저작권 관련 기관 자료

자동 수집 결과는 `research_insights.json`과 `memory/evolution_memory.json`에 저장됩니다.
이 자료는 생성 품질과 안전 체크를 개선하기 위한 것이며, 타인의 표현을 복제하기 위한 용도가 아닙니다.
