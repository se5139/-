카카오 이모티콘 수익화 시스템 v20 요약

핵심 업데이트: 누적 데이터 기반 성장형 추천 엔진

1. 20번 탭 '성장형 학습 엔진' 추가
2. 현재 세션의 캐릭터 분석, 표현 후보, 포맷 추천, 트렌드 결과, 품질검사, 채팅 미리보기, 저작권 방어, 샘플 세트 결과를 학습 스냅샷으로 누적 저장
3. 심사/판매 결과를 수동 기록 가능
4. 누적 데이터 기반 추천 포맷, 반복 키워드, 개선 액션, 데이터 건강 상태 리포트 생성
5. 기존 v19 데이터 보호/백업/마이그레이션 구조 유지
6. 학습 데이터는 append-only 방식으로 저장하며, 사용자 승인 없이 삭제하지 않는 구조

주의: 이 기능은 승인/수익 보장이 아니라 누적 데이터 기반 의사결정 보조 기능입니다.
