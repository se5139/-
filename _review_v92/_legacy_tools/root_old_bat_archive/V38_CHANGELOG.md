# v38 CSV/캡처 데이터 가져오기 + 학습 데이터 입력 파이프라인

## 핵심 추가
- CSV 템플릿 다운로드 기능 추가
- CSV 업로드/컬럼 자동 매칭/날짜·상태·숫자 검증 추가
- 캡처 이미지 원본 보존/미리보기/메모 기반 후보 추출 추가
- 카카오 심사 결과, 판매/정산 메모, 문구 반응, 반려 사유, 품질검사 결과, 트렌드 메모 데이터 유형 지원
- 원본 CSV/캡처와 정리본을 분리 저장
- 성장형 학습 JSONL에 append-only 방식으로 누적 저장
- 오류/경고 리포트와 전체 ZIP 생성

## 안전 원칙
- 기존 데이터를 덮어쓰지 않음
- 업로드 원본 보존
- 캡처 인식 결과는 사용자가 확인 후 저장
- API 자동수집은 보조, CSV/캡처 입력을 운영 데이터 중심으로 유지
