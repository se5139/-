# V39 변경 사항: 카카오 스튜디오 엑셀 자동 가져오기 / 성과 학습 엔진

## 추가 기능
- 카카오 이모티콘 스튜디오 `이모티콘 플러스 발신 통계` 엑셀 업로드 지원
- 카카오 이모티콘 스튜디오 `판매내역` 엑셀 업로드 지원
- XLSX 의존성 없는 zip/xml 기반 파서 추가
- 조회기간/판매기간 자동 추출
- 발신수/이용자수/반복률 계산
- 국내/일본/글로벌 판매 요약 정리
- 판매 상세 데이터 정리
- 작품별 성과 점수 계산
- 시리즈화/미니/움직이는 버전 확장 추천
- 원본 엑셀 보존, 정리 CSV/JSON 생성, 성장형 학습 JSONL 누적 저장

## 운영 원칙
- 데이터가 0이거나 비어 있으면 확장 판단보다 누적 저장 우선
- 처음부터 모든 포맷을 만들지 않고 v37 1차 포맷 전략과 연결
- 엑셀 원본은 덮어쓰지 않고 별도 보관
- 승인/수익 보장이 아닌 데이터 기반 추천 보조 기능
