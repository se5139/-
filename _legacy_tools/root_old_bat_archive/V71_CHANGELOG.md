# v71 제출 전 규격/용량/프레임 QC

## 핵심
- v70의 32개 정지형/24개 움직이는형 세트 후보를 제출 전 사전 QC로 점검합니다.
- 360×360 크기, PNG/GIF 형식, 투명 배경, 용량, GIF 프레임 수, GIF 후보 3개 이상, 파일명 정규화 계획을 검사합니다.
- Jinja2 HTML 리포트, QC CSV/JSON, 체크리스트 JSON, manifest, SQLite 학습 DB, QC ZIP 패키지를 생성합니다.
- 카카오 공식 기준은 변경될 수 있으므로 제출 직전 재확인 필요 문구를 명시합니다.

## 안전 원칙
- 기존 인기 캐릭터/문구/애니메이션 복제 금지.
- 온라인 자료는 추상 신호만 반영.
- API 키 원문은 리포트/ZIP/JSON/CSV에 저장하지 않음.
- 사용자 데이터 삭제/덮어쓰기 금지.
