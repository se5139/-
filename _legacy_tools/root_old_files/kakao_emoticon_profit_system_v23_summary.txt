카카오 이모티콘 수익화 시스템 v23 요약

이번 버전은 캐릭터 일관성 검사/자동 보정 기능을 추가했습니다.

추가 기능:
1. 24개/32개 세트 전체의 캐릭터 중심 위치 검사
2. 표현별 크기/표시 영역 일관성 검사
3. 대표 색상 차이 검사
4. 가장자리 잘림 위험 검사
5. 긴 문구로 인한 세트 톤 흔들림 경고
6. 표정/파츠 계획 분포 요약
7. 중심/크기/대표색 기준 자동 보정 미리보기 PNG 생성
8. HTML/JSON/CSV/ZIP 리포트 생성

기존 v22 기능은 유지됩니다.
- 제작 마법사
- 직접 그리기 캔버스/레이어 편집기
- 후보 갤러리
- 표정/파츠 편집기
- 채팅창 미리보기
- 제출 패키지
- 데이터 보호/백업
- 성장형 학습 엔진
